# Contributing to BlockRun

Thank you for your interest in contributing to BlockRun!

## How to Contribute

### 1. Report Bugs or Request Features

[Open an issue](https://github.com/blockrunai/awesome-blockrun/issues) with:
- Clear description of the bug or feature
- Steps to reproduce (for bugs)
- Expected vs actual behavior

### 2. Improve Documentation

- Fix typos or clarify existing docs
- Add examples or tutorials
- Translate documentation

### 3. Contribute Code

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Commit (`git commit -m 'Add amazing feature'`)
5. Push (`git push origin feature/amazing-feature`)
6. Open a Pull Request

### 4. Build Integrations

We're looking for integrations with:
- Agent frameworks (ElizaOS, LangChain, AgentKit)
- Trading bots (OctoBot, Freqtrade)
- Developer tools

See our [Integrations](./README.md#integrations) section for current status.

## Development Setup

cd awesome-blockrun

# For SDK development, see individual SDK repos:
