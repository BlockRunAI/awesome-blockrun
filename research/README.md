## State of x402 - December 2025

**63 million payments. $7.5M in USDC. 1,100+ projects.**

The first real-world proof that AI agents can pay for services at scale.

### Key Findings

- **Transaction Volume:** 63M+ transactions, $7.5M USDC
- **Average Transaction:** $0.12 (true micropayments)
- **Projects:** 1,100+ independent projects
- **Endpoints:** 4,800+ API endpoints
- **Network Split:** Base 53.3%, Solana 36.6%
- **Price Distribution:** 76% of services priced at $0.10 or below

### Methodology

Data collected from:
- x402scan.com transaction indexer
- 6 facilitator discovery APIs (Coinbase CDP, PayAI, Questflow, AnySpend, AurraCloud, thirdweb)
- Mainnet-only analysis (testnet/devnet filtered)
